## 📖 Sobre o projeto

O **Plano Semanal da Crisma** nasceu com o objetivo de auxiliar jovens da catequese a desenvolver uma rotina de oração, prática das virtudes e exame de consciência durante a preparação para o Sacramento da Crisma.

A proposta não é criar uma lista de tarefas ou um sistema de pontuação, mas oferecer um instrumento simples de acompanhamento espiritual, inspirado na ideia de uma **Regra de Vida**.

Todo o projeto foi desenvolvido para funcionar diretamente no navegador, sem necessidade de instalação, cadastro ou criação de conta.

---

## 🎯 Objetivos

* incentivar uma rotina de oração;
* estimular a prática semanal de uma virtude;
* favorecer momentos de reflexão pessoal;
* incentivar o exame diário de consciência;
* preservar a privacidade do jovem;
* funcionar completamente offline após carregado pelo navegador;
* manter simplicidade de uso.

# 🔒 Privacidade

Este projeto foi desenvolvido com a filosofia **Privacy First**.

Os dados do jovem:

* permanecem apenas no navegador;
* não são enviados para servidores;
* não são compartilhados com terceiros;
* não são enviados à catequista;
* não exigem login.

Quando implementado o backup, será possível exportar e importar os próprios dados manualmente.

# 💙 Filosofia do projeto

Este projeto não pretende medir espiritualidade.

O objetivo é apenas ajudar cada jovem a criar um pequeno espaço semanal para recordar sua intenção, cultivar uma virtude, rezar com constância e crescer em sua caminhada de fé.
